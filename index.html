A tag <p> é utilizada para criar parágrafos de texto.


Exemplo:

A tag a tag <mark> serve para destacar partes do texto, como se fosse um marca-texto, enquanto exibe o conteúdo em tamanho de fonte menor, útil para observações ou notas de rodapé.
Exemplos:
Tag <mark>: Este texto está destacado.
Tag <small>: Este texto está com um tamanho menor.

A tag <i> deixa o texto em itálico, geralmente para dar ênfase visual, sem significado semântico, e <u> sublinha o texto. Já a tag <strong> indica que o conteúdo é importante, aplicando um estilo em negrito com significado semântico.


Exemplos:
Tag <i>: Este texto está em itálico.
Tag <u>: Este texto está sublinhado.
Tag <strong>: Este texto está em negrito.

Listas

Para listas, a tag <ol> cria listas ordenadas (numeradas), enquanto <ul> define listas não ordenadas (com marcadores). Ambas usam a tag <li> para definir cada item da lista.


Exemplos:

Lista Ordenada

  1. Maça
  2. Uva
  3. Manga

Lista NÃO Ordenada

  • Front-end
  • Back-end
  • Full-stack

As tags <sub> e <sup> são usadas para subscrito (como em H₂O) e sobrescrito (como em 10²), respectivamente, úteis para fórmulas químicas ou matemáticas.

Exemplos:
Tag <sub>: CO2
Tag <sup>: 204

A tag <blockquote> é utilizada para exibir citações longas, geralmente com recuo em relação ao restante do texto.

Exemplo:
"a avaliação dos efeitos possíveis de um processo de judicialização das políticas dependerá da ponderação de diversos fatores: o grau de desenvolvimento dos sistemas de seguridade social e as capacidades de resposta do governo"

A tag <del> ndica texto que foi removido de um documento (normalmente aparece riscado). Enquanto, a <abbr> indica uma abreviação ou sigla, podendo mostrar o significado completo ao passar o mouse.


Exemplos:
Tag <del>: Texto removido
Tag <abbr>: ONU
